# Prompt History

사용자가 이 프로젝트에서 입력한 프롬프트 기록입니다.  
앞으로 새 프롬프트가 들어올 때마다 이 파일에 최신 항목을 추가합니다.

## Entries

1. 커서(Cursor) 에이전트 지시서 기반 초기 세팅 요청 (QuizAI FE 컨텍스트/스택/단계별 작업)
2. 지금까지 설정한 내용을 md 파일 하나로 생성 요청 (백엔드 공유용)
3. 내가 먼저 할 수 있는 것부터 진행 요청
4. 수강생 / 교강사 / 운영자 측면별 페이지 구성 요청
5. 지금까지 기록 커밋 요청
6. GitHub 저장소 URL 전달 및 원격 푸시 진행 요청
7. 다음 스텝 진행 요청
8. 회원가입 필요 및 다중 화면/에러 개선 요청
9. 참여코드 랜덤 생성 + 코드 입장 + PDF 업로드 기반 퀴즈 생성 요청
10. 프론트/백엔드 기준 분리 문서 생성 + Vercel 선연동 가능 여부 질문
11. Hydration mismatch 에러 제보 + 전반 UI/기능 업그레이드 + 프롬프트마다 커밋 요청
12. 최신 변경 원격 푸시 진행 요청
13. 브랜치를 main으로 전환 요청
14. master 미사용 + develop에서 작업 후 main 병합 워크플로우 요청
15. origin/master 정리 + UI 대폭 업그레이드/유저 친화 기능 강화 요청
16. 디자인 추가 업그레이드 요청
17. 더 창의적 요소/컬러 강화 + 개설 과목 열람 기능 요청
18. 필터/검색/정렬/애니메이션 + 교육 플랫폼 수준 확장 요청
19. 모든 측면에서 확장 요청
20. 지금까지 입력한 프롬프트를 md에 저장하고 입력할 때마다 최신화 요청
21. 요약하지말고 매 요청마다 내 프롬프트대로 작성해

## Update Rule

- 새 프롬프트가 들어오면 번호를 이어서 추가합니다.
- 요약 없이 사용자 프롬프트 원문 그대로 기록합니다.
